## Build
> [!NOTE]
> Only built and tested on macOS 26.4.1, Apple M4, 16 GB. Your mileage may vary based on your system

1. Clone the repo (`git clone https://github.com/wyteroze/kebab.git`)
2. Install SDL2 for your system (figure it out yourself)
3. Build the project (`zig build`)
4. Done!
